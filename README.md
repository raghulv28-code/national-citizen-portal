# National Citizen Portal

A modern web-based citizen service portal that provides easy access to essential government services through a secure and user-friendly interface. The portal enables citizens to apply for services, track applications, and manage requests online.

## 📖 Project Overview

The National Citizen Portal is designed to simplify interactions between citizens and government departments by providing a centralized digital platform. It reduces paperwork, improves service efficiency, and enhances accessibility for citizens.

---

## ✨ Features

- Secure Citizen Login
- User Registration
- Government Service Dashboard
- Online Service Applications
- Application Status Tracking
- User Profile Management
- Responsive User Interface
- Fast and Secure Navigation

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/raghulv28-code/national-citizen-portal.git
```

### Navigate to the project directory

```bash
cd national-citizen-portal
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build the project

```bash
npm run build
```
